# 📝 Заметочки

Приложение для управления заметками с авторизацией пользователей.  
Проект создан как практики полноценного взаимодействия между клиентом и сервером: работа с HTTP-запросами, авторизацией, формами, валидацией и обработкой состояний приложения.

---

# 🚀 Стек технологий

|Категория|Технологии|
|---|---|
|**Frontend**|React, TypeScript, Vite|
|**Backend**|Node.js, Express.js|
|**State Management**|React Context, Custom Hooks|
|**Forms & Validation**|react-hook-form, Zod|
|**Styling**|CSS Modules / CSS|
|**HTTP**|Fetch API, кастомный API-клиент|
|**Authentication**|JWT, HttpOnly Cookies, CORS|
|**Database**|Lowdb|

# ⚙️ Технические особенности

### ✅ Типобезопасность

- Полный TypeScript
- Дженерики
- Дискриминированный юнион
- Типизированные API-ответы
    

### ✅ Работа с формами

- `react-hook-form`
- `zod`
- `@hookform/resolvers`
    

### ✅ Управление состоянием
- `AuthContext` для авторизации
- Кастомный хук `useFetch`
- Конечный автомат состояний:
    
    ```txt
    idle → loading → success | error
    ```
    

### ✅ API-слой
- Абстракция `request()`
    
- Централизованная обработка ошибок
    
- Типизированные ответы сервера
    

### ✅ Архитектура

- Разделение на компоненты
    
- Переиспользуемые UI-компоненты
    
- Изоляция бизнес-логики

# 🚀 Быстрый старт

## 1. Установка проекта

```bash
git clone https://github.com/AtinsS/Notes.git
cd notes-app
npm i
```

---

## 2. Запуск в режиме разработки

### Frontend

```bash
cd client
npm run dev
```

Frontend запускается через Vite на:

```txt
http://localhost:5173
```

### Backend

```bash
cd server
npm run dev
```

Backend запускается через Express.js на:

```txt
http://localhost:4000
```
